DEFINICION DE ETIQUETAS EN HTML
!DOCTYPE html:
Al navegador que el documento está basado en el estándar HTML5
TITLE:
Usada para definir el título de la página web.
META:
Usada para definir otros metadatos que no se pueden definir con una etiqueta HTML especifica. Por ejemplo, para definir el autor del sitio, o la descripción del mismo.
STYLE:
usadas para introducir codigo CSS en línea, es decir, en el propio documento HTML.
NAV:
Usadas para definir el contenido que será la navegaciónsección de de la web.
MAIN:
Se usa para definir el contenido principal del documento. Solamente puede existir uno por documento.
SECTION:
Define una seccion sección del documento.
ARTICLE:
Article, /article Define contenidoindependiente de la web.
ASIDE:
De estas etiquetas suele alojarse el contenido adicional de la web. Suele ser contenido relacionado con la web pero de poca importancia
FOOTER :
footer Usadas para definir el pie de pagina.
